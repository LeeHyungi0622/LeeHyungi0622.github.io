210505 screen.getBy*мҷҖ getByмқҳ м°Ёмқҙ, findBy*/getBy*/queryBy* methodмқҳ м°Ёмқҙ


н…ҢмҠӨнҠё мҪ”л“ңлҘј мһ‘м„ұн•ҳлҚҳ лҸ„мӨ‘м—җ н—·к°Ҳл ёлҚҳ к°ңл…җл“Өм—җ лҢҖн•ҙм„ң л°ҳліөн•ҷмҠөмқ„ мң„н•ҙ нҸ¬мҠӨнҢ…мқ„ н•ҙл‘җл Өкі  н•ңлӢӨ.

screen.getBy vs getBy

screenмқ„ мӮ¬мҡ©н•ҳлҠ” мқҙм җмқҖ н•„мҡ”н•ң queryлҘј м¶”к°Җн•  л•Ң лҚ” мқҙмғҒ render н•ЁмҲҳлЎң м»ҙнҸ¬л„ҢнҠёлҘј л ҢлҚ”л§Ғн•ҙм„ң мөңмӢ  мғҒнғңлҘј мң м§Җ н•ҳм§Җ м•Ҡм•„лҸ„ лҗңлӢӨлҠ” м җмқҙлӢӨ.
beforeEach()н•ЁмҲҳ лӮҙм—җм„ң кіөнҶөм ҒмңјлЎң м»ҙнҸ¬л„ҢнҠёлҘј л ҢлҚ”л§Ғн–ҲлӢӨл©ҙ, мқҙн•ҳмқҳ test лё”лҹӯм—җм„ңлҠ” screenмқ„ нҶөн•ҙ л ҢлҚ”л§Ғлҗң м»ҙнҸ¬л„ҢнҠёлҘј нҶөн•ҙ н…ҢмҠӨнҠё мҪ”л“ңлҘј мһ‘м„ұн•  мҲҳ мһҲлӢӨ.

findBy* / getBy* / queryBy* methodмқҳ м°Ёмқҙ

  • getBy*

    мЎ°кұҙм—җ л§һлҠ” мҡ”мҶҢлҘј м°ҫм•ҳмқ„л•Ң queryм—җ мқјм№ҳн•ҳлҠ” л…ёл“ңлҘј л°ҳнҷҳ
    мЎ°кұҙм—җ л§һлҠ” мҡ”мҶҢлҘј лӘ» м°ҫм•ҳмқ„л•Ң м—җлҹ¬лҘј л°ҳнҷҳ
    вҶ’ мЎҙмһ¬н•ҳм§Җ м•ҠлҠ” DOM elementлҘј assertн• л•Ңм—җлҠ” try-catchл¬ёмңјлЎң л°ңмғқн•ң м—җлҹ¬лҘј лі„лҸ„мқҳ н•ЁмҲҳм—җ лӢҙм•„ tobedefined() matcherлЎң нҷ•мқёмқ„ н•ҙм•ј н•ңлӢӨ. мқҙ кІҪмҡ°м—җлҠ” queryBy*лҘј мӮ¬мҡ©н•ҙм„ң assertн•ҳлҠ” кІғмқҙ мўӢлӢӨ.

  • queryBy*

    мЎ°кұҙм—җ л§һлҠ” мҡ”мҶҢлҘј м°ҫм•ҳмқ„л•Ң queryм—җ мқјм№ҳн•ҳлҠ” л…ёл“ңлҘј л°ҳнҷҳ
    мЎ°кұҙм—җ л§һлҠ” мҡ”мҶҢлҘј лӘ» м°ҫм•ҳмқ„л•Ң nullмқ„ л°ҳнҷҳ
    вҶ’ мЎҙмһ¬н•ҳм§Җ м•ҠлҠ” DOM elementлҘј assertн• л•Ң мң мҡ©н•ҳлӢӨ.

  • findBy*

    мЎ°кұҙм—җ л§һлҠ” мҡ”мҶҢлҘј м°ҫм•ҳмқ„л•Ң resolved Promise л°ҳнҷҳ
    мЎ°кұҙм—җ л§һлҠ” мҡ”мҶҢлҘј лӘ» м°ҫм•ҳмқ„л•Ң rejected Promise л°ҳнҷҳ

    кё°мЎҙмқҳ waitForElement()н•ЁмҲҳлҠ” мқҙм ң deprecatedлҗҳм—Ҳкё° л•Ңл¬ём—җ findBy* н•ЁмҲҳлҘј мӮ¬мҡ©н•ҙм•ј н•ңлӢӨ.

210429 React TIL - н”„лЎ нҠём—”л“ң к°ңл°ңмӢң dummy лҚ°мқҙн„° нҷңмҡ©, Container - Presenter кө¬мЎ°, Re-renderingкіј нғңк·ёлӮҙм—җм„ңмқҳ мқёлқјмқё мҠӨнғҖмқјл§Ғ, н•ЁмҲҳнҳ• м»ҙнҸ¬л„ҢнҠём—җм„ңмқҳ лҰ¬л ҢлҚ”л§Ғ, eslint м„Өм •

ліё нҸ¬мҠӨнҢ… лӮҙмҡ©мқҖ кіјкұ°м—җ к°ңмқём ҒмңјлЎң кіөл¶Җн• л•Ң м •лҰ¬н–ҲлҚҳ ReactJSмқҳ лӮҙмҡ©мқ„ ліөмҠөмқҳ лӘ©м ҒмңјлЎң лӢӨмӢң м •лҰ¬н•ҳлҠ” нҸ¬мҠӨнҢ…мһ…лӢҲлӢӨ.

н”„лЎ нҠём—”л“ң к°ңл°ңмӢң dummy лҚ°мқҙн„° нҷңмҡ©

л§Ңм•Ҫ л°ұм—”л“ңк°Җ к°ңл°ңмқҙ м•Ҳлҗң мғҒнғңм—җм„ң н”„лЎ нҠём—”л“ң к°ңл°ңмқ„ м§„н–үн•ҙм•ј лҗҳлҠ” кІҪмҡ°лқјл©ҙ лҚ”лҜё лҚ°мқҙн„°лҘј нҷңмҡ©н•ҙм„ң мҡ°м„ м ҒмңјлЎң н”„лЎ нҠём—”л“ң к°ңл°ңмқ„ н•ҳлҸ„лЎқ н•ңлӢӨ.
мҳҲлҘјл“Өм–ҙ лЎңк·ёмқё нҷ”л©ҙмқ„ л§Ңл“ңлҠ”лҚ°, лЎңк·ёмқёкіј лЎңк·ём•„мӣғл“ұ мғҒнҷ©м—җ л”°лҘё лӢӨлҘё н”„лЎ нҠё нҺҳмқҙм§ҖлҘј л§Ңл“Өм–ҙм•ј н•ңлӢӨл©ҙ, мқҙ кІҪмҡ°м—җ к°„лӢЁн•ҳкІҢ лҚ”лҜёлЎң мғҒнғң лҚ°мқҙн„°лҘј л§Ңл“Өм–ҙм„ң нҷ”л©ҙмқ„ л§Ңл“ӨлҸ„лЎқ н•ңлӢӨ.
мӢӨл¬ҙм—җм„ңлҠ” ReduxлӮҳ MobxлҘј мӮ¬мҡ©н•ҙм„ң dummy stateлҘј л§Ңл“ лӢӨ.

Container - Presenter кө¬мЎ°

мқҙм „м—җ нҒҙлһҳмҠӨ м»ҙнҸ¬л„ҢнҠёлЎң мһ‘м„ұмқ„ н–Ҳмқ„л•ҢлҠ” лҚ°мқҙн„°лҘј л°ӣм•„м„ң лЎңм§Ғмқ„ мІҳлҰ¬н•ҳлҠ” л¶Җл¶„(Container)кіј нҷ”л©ҙм—җ м¶ңл Ҙн•ҙмЈјлҠ” л¶Җл¶„(Presenter)мқ„ лӮҳлҲ м„ң кө¬мЎ°лҘј м§°лҠ”лҚ°, hooksк°Җ лҸ„мһ…лҗң мқҙнӣ„м—җлҠ” мң„мҷҖк°ҷмқҙ кө¬мЎ°лҘј лӮҳлҲ„лҠ” кІғмқ„ м¶”мІңн•ҳм§Җ м•ҠлҠ”лӢӨкі  н•ңлӢӨ.

л°ҳліөлҗҳлҠ” мһ‘м—…мқҖ мІҳмқҢл§Ң мҲҳмһ‘м—…мңјлЎң, к·ё лӢӨмқҢл¶Җн„°лҠ” лқјмқҙлёҢлҹ¬лҰ¬ мӮ¬мҡ©

мҲҳмһ‘м—…мңјлЎң л§Ңл“ңлҠ” н•ҳлӮҳ н•ҳлӮҳ л§Ңл“Өм–ҙліҙлҠ” кІғлҸ„ мқҳлҜёк°Җ мһҲлӢӨ. н•ҳм§Җл§Ң л§ӨлІҲ н•ҳлӮҳ н•ҳлӮҳ л§Ңл“Өм–ҙліҙлҠ” кІғліҙлӢӨлҠ” лқјмқҙлёҢлҹ¬лҰ¬лҘј мӮ¬мҡ©н•ҙліҙлҠ” м—°мҠөмқ„ н•ҳлҠ” кІғлҸ„ мӨ‘мҡ”н•ҳлӢӨ. мӢӨм ң мӢӨл¬ҙм—җм„ң лӘЁл“  кІғмқ„ мҲҳмһ‘м—…мңјлЎң л§Ңл“Өм§ҖлҠ” м•Ҡкё° л•Ңл¬ёмқҙлӢӨ.

мһҗмӢқ componentм—җ propsлЎң л„ҳкІЁмЈјлҠ” н•ЁмҲҳмқҳ кІҪмҡ°м—җлҠ” л°ҳл“ңмӢң useCallbackмңјлЎң к°җмӢём„ң мІҳлҰ¬н•ңлӢӨ.
