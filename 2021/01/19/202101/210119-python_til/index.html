Python

мҳӨлҠҳ л°°мҡҙлӮҙмҡ©

  • Pythonмқҳ кё°ліё мһҗлЈҢнҳ•
    • string
    • list
    • number
    • bool
    • int
    • float
  • Pythonмқҳ лӮҙмһҘн•ЁмҲҳ
    • print()
    • type()
    • round() : л°ҳмҳ¬лҰјн•ҳкі мһҗ н•ҳлҠ” к°’кіј мҶҢмҲ«м җмқҙн•ҳ NлІҲм§ё мһҗлҰ¬к№Ңм§Җ мҲҳлҘј н‘ңмӢңн•ҳкё° мң„н•ҙм„ң мқёмһҗ Nмқ„ л„Јм–ҙмӨҖлӢӨ.
  • Pythonмқҳ мҲ«мһҗ мһҗлЈҢнҳ•кіј м—°мӮ°
    • мӮ¬м№ҷм—°мӮ°(+,-,*,/)
    • // (лӘ« кө¬н•ҳлҠ” м—°мӮ°мһҗ), % (лӮҳлЁём§Җ кө¬н•ҳлҠ” м—°мӮ°мһҗ)
    • ** (м§ҖмҲҳ м—°мӮ°мһҗ)
  • Python ліҖмҲҳмқҳ лҚ°мқҙн„° нғҖмһ…кІҖмӮ¬
    • ліҖмҲҳмқҳ лҚ°мқҙн„° нғҖмһ…кІҖмӮ¬лҠ” type() лӮҙмһҘн•ЁмҲҳлҘј мӮ¬мҡ©н•ҳм—¬ мІҙнҒ¬ н•  мҲҳ мһҲлӢӨ.
  • for-loopлҘј нҷңмҡ©н•ң л°ҳліөм—°мӮ°
    • for - in - range() кө¬л¬ёмқ„ нҷңмҡ©н•ҳм—¬ мқјм • кө¬к°„м—җ мһҲлҠ” мҲ«мһҗлҘј мҲңнҡҢн•ҳл©° мІҳлҰ¬н•  мҲҳ мһҲлӢӨ.

к°„лӢЁн•ң м—°мҠөл¬ём ң

мҳӨлҠҳ л°°мҡҙ м „л°ҳм Ғмқё лӮҙмҡ©мқ„ нҷңмҡ©н•ҳм—¬ м•„лһҳ л¬ём ңлҘј н•ҙкІ°н•ңлӢӨ.

л°ҳм§ҖлҰ„(r=10)мқ„ м„ м–ён•ң л’Ө, мқҙлҘј мқҙмҡ©н•ҳм—¬ мӣҗмқҳ м§ҖлҰ„, л‘ҳл Ҳ, л„“мқҙ, кө¬мқҳ кІүл„“мқҙ, л¶Җн”јлҘј к°Ғк°Ғ м¶ңл Ҙн•ҳлҠ” нҢҢмқҙмҚ¬ нҢҢмқјмқ„ л§Ңл“Өм–ҙліҙм„ёмҡ”.(pi=3.1415)

sample output

r(л°ҳм§ҖлҰ„)мқҳ кёёмқҙк°Җ 10мқјл•Ңмқҳ м¶ңл Ҙ sample

r = 10 ==> print("r =", r)
d = 20
c = 62.830
a = 314.15
gnb = 1256.0000
v = 4188.666666666667

л¬ём ңн’Җмқҙ

# л°ҳм§ҖлҰ„кіј piк°’мқ„ м„ м–ён•ңлӢӨ.
r = 10
pi = 3.1415

#  d: м§ҖлҰ„, c: л‘ҳл Ҳ, a: л„“мқҙ
# gnb: кө¬мқҳ л„“мқҙ, v: л¶Җн”ј
d = 2 * r
c = 2 * pi * r
a = pi * r ** 2
gnb = 4 * pi * r ** 2
v = 4 / 3 * pi * r ** 3

print("r =", r)
print("c =", round(c, 4))
print("a =", a)
print("gnb =", round(gnb, 2))
print("v =", round(v,4))
