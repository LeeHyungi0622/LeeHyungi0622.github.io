  • Pandas

  • Pandas?

    • pandas는 Python에서 데이터 분석 기능을 제공하는 라이브러리이다. 특히 몇 테이블 및 시계열 데이터를 조작하기위한 데이터 구조와 연산을 제공한다. Pandas는 BSD License하에 제공되고있다.

    여기서 BSD License란, BSD(Berkeley Software Distribution)로, 자유 소프트웨어 저작권의 한 가지로 분류된다.
