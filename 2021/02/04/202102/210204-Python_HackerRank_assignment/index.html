Problem1 Link(HackerRank)

мқҙ л¬ём ңлҠ” м•һ л’ӨлЎңлҠ” мһҗмқҢл§Ң мң„м№ҳн•ҳкі , к°ҖмҡҙлҚ°м—җлҠ” 2к°ң мқҙмғҒмқҳ лӘЁмқҢмңјлЎңл§Ң кө¬м„ұлҗҳм–ҙ мһҲлҠ” л¬ёмһҗлҘј м •к·ңн‘ңнҳ„мӢқмқ„ нҶөн•ҙ м¶ңл Ҙн•ҳлҠ” л¬ём ңмқҙлӢӨ.

Task You are given a string . It consists of alphanumeric characters, spaces and symbols(+, -). Your task is to find all the substrings of that contains or more vowels. Also, these substrings must lie in between consonants and should contain vowels only.

мҡ°м„  л¬ём ңлҘј н’Җмқҙн•ҳкё°м—җ м•һм„ң лӢӨмқҢ кёҚм •/л¶Җм •мҳҲмёЎ(positive/negative lookahead assertion), кёҚм • нӣ„ мқҪкё°м—җ лҢҖн•ң к°ңл…җм—җ лҢҖн•ҙм„ң м•Ңм•„ліҙмһҗ.

кёҚм •м Ғмқё мҳҲмёЎ

x?=y : мқҙ нҢЁн„ҙмқҖ м§Ғнӣ„м—җ yк°Җ мЎҙмһ¬н•ҳлҠ” л¬ёмһҗм—ҙ xм—җ matchн•ңлӢӨ.

foo(?=bar)
# м§Ғнӣ„м—җ barк°Җ мһҲлҠ” fooм—җ мқјм№ҳн•ңлӢӨ.

л¶Җм •м Ғмқё мҳҲмёЎ

x?!y : мқҙ нҢЁн„ҙмқҖ м§Ғнӣ„м—җ yк°Җ м—ҶлҠ” л¬ёмһҗм—ҙ xм—җ matchн•ңлӢӨ.

foo(?!bar)
# м§Ғнӣ„м—җ barк°Җ м—ҶлҠ” fooм—җ мқјм№ҳн•ңлӢӨ.

кёҚм • нӣ„ мқҪкё°

x?<=y : мқҙ нҢЁн„ҙмқҖ м§Ғм „м—җ л¬ёмһҗм—ҙ xк°Җ мһҲлҠ” л¬ёмһҗм—ҙ y(xлҠ” нҸ¬н•Ён•ҳм§Җ м•ҠмқҢ)м—җ мқјм№ҳн•ңлӢӨ.

(?<=bar)foo
# м§Ғм „м—җ barк°Җ мһҲлҠ” foo(barлҠ” нҸ¬н•Ён•ҳм§Җ м•ҠмқҢ)м—җ мқјм№ҳн•ңлӢӨ.

л¶Җм • нӣ„ мқҪкё°

x?<!y : мқҙ нҢЁн„ҙмқҖ м§Ғм „м—җ xк°Җ м—ҶлҠ” л¬ёмһҗм—ҙ yм—җ мқјм№ҳн•ңлӢӨ.

(?<!bar)foo
# м§Ғм „м—җ barк°Җ м—ҶлҠ” foo(barлҠ” нҸ¬н•Ён•ҳм§Җ м•ҠмқҢ)м—җ мқјм№ҳн•ңлӢӨ.

ліё мҪ”л“ң

import re

result = re.findall(
r'(?<=[QWRTYPSDFGHJKLZXCVBNMqwrtypsdfghjklzxcvbnm])([aeiouAEIOU]{2,})(?=[QWRTYPSDFGHJKLZXCVBNMqwrtypsdfghjklzxcvbnm])', input())

if result:
    print(*result, end='\n')
else:
    print(-1)
find_object = re.finditer(
    r'(?<=[QWRTYPSDFGHJKLZXCVBNMqwrtypsdfghjklzxcvbnm])([aeiouAEIOU]{2,})(?=[QWRTYPSDFGHJKLZXCVBNMqwrtypsdfghjklzxcvbnm])', input()
)

for i in map(lambda x: x.group(), find_object):
    print(i)

Problem2 Link(HackerRank)

мқҙ л¬ём ңлҠ” мһ…л Ҙл°ӣмқҖ multiline statementsм—җм„ң к°Ғ л¬ёмһҗм—ҙм—җ &&к°Җ мһҲмңјл©ҙ andлЎң, ||к°Җ мһҲмңјл©ҙ orлЎң ліҖнҷҳн•ҳлҠ” л¬ём ңмҳҖлӢӨ.
мқҙ л¬ём ңлҘј н’Җл•Ң мІҳмқҢм—җлҠ” [\s]лЎң м•һ/л’Ө кіөл°ұмқ„ кө¬л¶„н•ҳл Өкі  н–ҲлҠ”лҚ° мҳҲмғҒ кІ°кіјк°’мңјлЎң м¶ңл Ҙмқҙ м•Ҳлҗҳм–ҙ, ((?<=[white space]), (?=[white space]))лЎң лӢӨмӢң м •к·ңн‘ңнҳ„мқ„ мһ‘м„ұн•ҳм—¬ н’Җмқҙн•ҳмҳҖлӢӨ.

You are given a text of lines. The text contains && and || symbols.
Your task is to modify those symbols to the following:
Both && and || should have a space " " on both sides.

&& вҶ’ and
|| вҶ’ or

ліё мҪ”л“ң

import re

def cvtLogicalOperator(match):
    if match.group(0) == "||":
        return "or"
    elif match.group(0) == "&&":
        return "and"


N = int(input())
statement = []
for _ in range(N):
    statement.append(input())

statement = '\n'.join(statement)

print(re.sub(r"(?<= )(&&|\|\|)(?= )", cvtLogicalOperator, statement))

Problem3 Link(HackerRank)

мқҙ л¬ём ңлҠ” мһ…л Ҙл°ӣмқҖ мқҙл©”мқј мЈјмҶҢлҘј м •к·ңн‘ңнҳ„мӢқ(regular expression)мқ„ мӮ¬мҡ©н•ҙм„ң мң нҡЁм„ұ кІҖмӮ¬лҘј н•ҳлҠ” л¬ём ңмқҙлӢӨ.
мқҙ л¬ём ңм—җм„ңлҠ” мқҙм „м—җ л°°мҡҙ meta characterлҘј лӢӨм–‘н•ҳкІҢ мӮ¬мҡ©н•ҙ ліј мҲҳ мһҲлӢӨ.

The first line contains a single integer, , denoting the number of email address.
Each line of the subsequent lines contains a name and an email address as two space-separated values following this format:

name <user@email.com>

ліё мҪ”л“ң

import re
import email.utils

n = int(input())

for _ in range(n):
    name, email = map(str, input().split())
    # < нҠ№мҲҳл¬ёмһҗлЎң ^(мӢңмһ‘)н•ҳкі , \w(alphanumeric character),
    # .мқ„ л¬ёмһҗ мһҗмІҙлЎң мһ…л Ҙн•ҳкё° мң„н•ҙ \(back slash)мҷҖ к°ҷмқҙ мһ…л Ҙ,
    # лҒқмқҖ >лЎң лҒқлӮҳкё° л•Ңл¬ём—җ лҒқ л¬ёмһҗ л’Өм—җ $ л©”нғҖл¬ёмһҗлҘј мһ…л Ҙн•ҙм„ң мІҳлҰ¬н•ҙмӨҖлӢӨ.
    ptn = r"^<[a-zA-Z](\w|-|\.|_)+@[a-zA-Z]+\.+[a-zA-Z]{1,3}>$"
    if re.match(ptn, email):
        print(name, email)

мқҙл¬ём ңлҠ” мһ…л Ҙл°ӣмқҖ м „нҷ”лІҲнҳёлҘј м •к·ңн‘ңнҳ„мӢқмқ„ мӮ¬мҡ©н•ҙм„ң мң нҡЁм„ұ кІҖмӮ¬лҘј н•ҳлҠ” л¬ём ңмқҙлӢӨ.
л¬ём ңм—җм„ң мһ…л Ҙл°ӣлҠ” м „нҷ”лІҲнҳёмқҳ мЎ°кұҙмқҖ м•„лһҳмҷҖ к°ҷлӢӨ.

condition1) м „нҷ”лІҲнҳёлҠ” мҙқ 10мһҗлҰ¬ мҲҳлЎң кө¬м„ұмқҙ лҗңлӢӨ.
condition2) м•һ мһҗлҰ¬лҠ” 7, 8, 9лЎң мӢңмһ‘н•ңлӢӨ.
condition3) лӮҳлЁём§Җ мһҗлҰ¬мҲҳлҠ” мҲ«мһҗ(\d)лЎң кө¬м„ұлҗңлӢӨ.

ліё мҪ”л“ң

test_case = int(input())
for _ in range(test_case):
    phone_number = input()
    ptn = r"[7-9][0-9]{9}"
    print('YES' if len(phone_number) == 10 and re.match(ptn, phone_number) else 'NO')
