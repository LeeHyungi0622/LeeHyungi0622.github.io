타입별칭

일반적으로 변수를 선언하듯이 type 키워드로 type을 선언해서 사용할 수 있다.

```javascript
type Age = number
type persono = {
  name: string
  age: Age
}
```
```javascript
let age: Age = 55
let driver: Person = {
  name: 'James May',
  age: age
}
```
