210202 ExpressJS Installation

2021-02-02

ліё нҸ¬мҠӨнҢ… лӮҙмҡ©мқҖ кіјкұ°м—җ к°ңмқём ҒмңјлЎң кіөл¶Җн• л•Ң м •лҰ¬н–ҲлҚҳ NodeJSмқҳ лӮҙмҡ©мқ„ ліөмҠөмқҳ лӘ©м ҒмңјлЎң лӢӨмӢң м •лҰ¬н•ҳлҠ” нҸ¬мҠӨнҢ…мһ…лӢҲлӢӨ.


ExpressJS м„Өм№ҳ

NPM(Node Package Manager) : м—…лҚ°мқҙнҠёлҗң к°Ғмў… NodeJS кҙҖл Ё нҢЁнӮӨм§Җмқҳ мӨ‘м•ҷм§‘мӨ‘нҷ”лҗң к°ңл…җмңјлЎң NodeJS мӣ”л“ңмқҳ мӨ‘мӢ¬кіј к°ҷмқҖ кіімқҙлӢӨ.

NPMмқҖ NodeJSлҘј лӢӨмҡҙл°ӣкІҢ лҗҳл©ҙ мһҗлҸҷмңјлЎң м„Өм№ҳк°Җ лҗҳкё° л•Ңл¬ём—җ лі„лҸ„лЎң м„Өм№ҳн•  н•„мҡ”лҠ” м—ҶлӢӨ.

Package managerлЎң npmмқ„ мӮ¬мҡ©н•ҳкё° мң„н•ҙм„ңлҠ” мҡ°м„  npmмқҙ м •н•ң л°©мӢқмңјлЎң н”„лЎңм қнҠёлҘј мӢңмһ‘н•ҙм•јн•ңлӢӨ.

  1. npm init : npm мҙҲкё°нҷ” (package name/description/author м„Өм •)
    вҶ’ кІ°кіјм ҒмңјлЎң package.json нҢҢмқјмқҙ мғқм„ұмқҙ лҗңлӢӨ.

  2. npm м„Өм№ҳ : вҖҳnpm install expressвҖҷ лӘ…л №мӢӨн–үн•ңлӢӨ.
    (package.json нҢҢмқјмқҙ мһҲлҠ” нҸҙлҚ”мң„м№ҳм—җм„ң мӢӨн–үмқ„ н•ҙм•јн•ңлӢӨ)

  3. м„Өм№ҳк°Җ лҒқлӮңл’Өм—җ мӮҙнҺҙліҙм•„м•јн•ҳлҠ” кІғмқҖ node modules л¶Җл¶„мқҙлӢӨ. мқҙ л¶Җл¶„мқҖ npmмқ„ нҶөн•ҙм„ң лӢӨмҡҙмқ„ л°ӣмқҖ кІғл“ӨмқёлҚ° м—¬кё°ліҙл©ҙ dependencyлқјлҠ” кІғмқ„ к°–кІҢ лҗңлӢӨ.
    м–ҙн”ҢлҰ¬мјҖмқҙм…ҳ к°ңл°ңмӢңм—җ н•„мҡ”н•ң кІғл“Өмқ„ мһҗлҸҷмңјлЎң npm moduleм—җм„ң м°ҫм•„м„ң мӮ¬мҡ©н•ҙмӨҖлӢӨ.

н”„лЎңм қнҠё нҳ‘м—…мӢңм—җлҠ” мһ‘м„ұн•ң мҪ”л“ңмҷҖ Package.json нҢҢмқјл§Ң л„ҳкІЁмЈјкё°

  • л‘җ нҢҢмқјл§Ң л°ӣм•„м„ң н”„лЎңм қнҠё нҸҙлҚ” лӮҙл¶Җм—җм„ң npm installл§Ң мӢӨн–үн•ҙмЈјл©ҙ мһҗлҸҷмңјлЎң н•„мҡ”н•ң нҢҢмқјл“Өмқ„ лӢӨмӢң м „л¶Җ мғқм„ұн•ҙмӨҖлӢӨ.
