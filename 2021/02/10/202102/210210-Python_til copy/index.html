얕은복사, 깊은복사

  • 참조변수에 대해서 type casting을 하게 되면 참조한 변수에 대해서 새로운 객체로 참조된다.
fruits = ['apple']

a = fruits
print(a) #['apple']

b = fruits
b.append('banana')
print(b) #['apple', 'banana']

print(fruits) #['apple', 'banana']

fruits 변수가 b에서 업데이트된 값으로 초기화가 되었다.
이러한 경우는 얕은 복사로, 깊은 복사를 해주기 위해서는 아래와 같이 작성을 해줘야 한다.

fruits = ['apple']

a = list(fruits)
print(a) #['apple']

b = list(fruits)
b.append('banana')
print(b) #['apple', 'banana']

print(fruits) #['apple', 'banana']

class내의 method에 반복처리하는 구문 처리하기

class Hero():
  def save_item_multiple(self, item, num):
    for _ in range(num):
      self.save_item(item)

  def save_items(self, *items):
    for item in items:
      self.save_item(item)
      print('{} has saved to inventory'.format(item))

IronMan = Hero('iron man', 'suit')
IronMan.inventory
# 반복하는 처리를 할 수 있다.
IronMan.save_items('$10,000', '2btc', '3eth', '10000 xrp')
