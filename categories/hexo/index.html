Hexo 사용법 Hexo에 파일을 수정/업데이트 한 뒤에 사용하는 명령어에 대해서 정리 hexo generate는 로컬 환경에서 Hexo 페이지의 변경된 내용을 확인할 때 사용되며, hexo deploy는 업데이트 된 내용을 Git에 최종적으로 deploy함으로써 Github pages에 변경사항을 적용한다. $ hexo clean &amp;&amp; hexo generate $ hexo clean &amp;&amp; hexo deploy
